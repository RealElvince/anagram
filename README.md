# anagram

## Anagram Algorithm Pseudocode

1. Start
2. Input first_word
3. Input second_word
4. Convert both words to lowercase
5. If length of first_word is not equal to length of second_word
      Print "Not anagrams"
6. Else
      Sort characters of first_word
      Sort characters of second_word
      If sorted first_word equals sorted second_word
            Print "Words are anagrams"
      Else
            Print "Words are not anagrams"
7. End
